## Build Setup

**Requirement**

- [Node.js](https://nodejs.org/en/)

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run build && npm run export
# serve static project
$ npm run serve
```
